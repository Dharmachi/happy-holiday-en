# 快乐假期 · Happy Holiday

初二英语游戏化学习 App（简体中文界面）。当前已收录 **Unit 1 Happy Holiday**。

## 本地打开

用浏览器打开 `index.html`，或在本目录启动本地服务：

```bash
python3 -m http.server 5173
```

然后访问 http://localhost:5173

手机可用同一 Wi‑Fi 下的电脑 IP 访问；Safari / Chrome 可「添加到主屏幕」。

## 已有游戏

- 单词闪卡 / 中英互译 / 配对 / 拼写
- 短语闯关
- 不定代词语法
- 课文挖空 / 事件排序 / 句子重组 / 课文朗读
- 错题本 + 本地星星进度
